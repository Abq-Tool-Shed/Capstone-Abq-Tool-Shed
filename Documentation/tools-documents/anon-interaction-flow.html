Anonymous Browser Interaction flow

1. The anonymous user loads the ABQ Tool Shed landing page.

2. The anonymous user scrolls through the page to see whether he can find the tool he needs.

3. The anonymous user realizes that they need to sign up to access the tool listings.

4. The anonymous user browses throughout the website to determine if they should sign up.

5. The anonymous user reads our mission statement and FAQ which are on the landing page.

6. The user determines its worth signing up and clicks the sign-up button.

7. The user is prompted to enter the necessary registration info.

8. The user is registered and browsing for the tool they need.
