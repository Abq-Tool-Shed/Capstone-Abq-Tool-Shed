Anonymous Browser Interaction flow

1. The anonymous user loads the ABQ Tool Shed landing page.

1a. The server responds to the request and grants access to the site.

2. The anonymous user scrolls through the page to see whether he can find the tool he needs.

2a. The server loads the newest listings.

3. The anonymous user finds their tool in our featured tool section but realizes that they need to sign up to access the tool listings.

3a. The user is sent to the sign-up page whenever they click on the tool.

4. The anonymous user goes back to browse throughout the website and determine if they should sign up.

4a. The user accomplishes this by clicking the home link in the nav bar at the top.

5. The anonymous user reads our mission statement and FAQ which are on the landing page.

6. The user determines its worth signing up and clicks the sign-up button.

6a. The user is redirected to the sign-up page.

7. The user is prompted to enter the necessary registration info.

7a. The user enters their information in our sign-up page and the data is stored in our database.

8. The user is registered and is taken to an email confirmation screen.

8a. Once the user confirms the email we receive confirmation and activate their account.

9. The user clicks the home button in the top nav bar and is sent back to the logged-in home screen.


User Story

"I am looking to do some lawn care, but I don't have the tools that I need. I don't quite have the cash to buy the tools on my own but my friend recommended me a website where someone could lend me a tool for the time being."
