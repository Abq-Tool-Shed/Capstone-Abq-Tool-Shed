Relationships

Profiles to Tools: 1 to many
Each tool is for one profile only (no splitting in our example)
One profile can add multiple tool listings
Borrow to Tools: many to many
Each tool can be borrowed by one profile at a time
many profiles can borrow multiple tool at a time
Rating to Profile: many-to-many-to-one
Many profiles can rate many profiles
The can only be one rating per borrow interaction
